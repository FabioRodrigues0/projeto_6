# Calculadoras

## Objetivo
Pretende-se desenvolver um programa em Java que represente que permita a utilização de diferentes tipos de calculadoras.

## Objetivos Pedagógicos
- Aplicar conceitos de classes e objetos
- Utilizar encapsulamento
- Praticar sobrecarga de métodos
- Utilizar diferentes packages no mesmo projeto

## Desenvolvimento do Projeto
- Criar um novo projeto, designado por “calculos”.
- Dentro do projeto criado, criar um novo package com o nome “calculadoras”.
- No package “calculadoras” criar uma classe com o nome “Calculadora”.
- Criar na classe “Calculadora” métodos de instância para executar as operações aritméticas soma, subtração, multiplicação, divisão inteira, resto da divisão inteira e divisão real. Os operandos são números reais.
- Na classe existente no package “calculos”, criar uma interface baseada num menu de opções para utilizar as operações aritméticas definidas na classe “Calculadora”:
    - (1) Inserir operandos
    - (2) Soma
    - (3) Subtração
    - (4) Multiplicação
    - (5) Divisão
    - (6) Divisão Inteira
    - (7) Resto da Divisão Inteira
    - (8) Sair
- No programa principal (método main da classe criada no package “calculos”), criar um objeto da classe “Calculadora” que permita executar as operações existentes no menu.
- Criar uma nova classe no packge “calculadoras”, a designar por “CalculadoraCientifica” que deverá disponibilizar as mesmas operações existentes na classe “Calculadora”, acrescentando as seguintes: raiz quadrada, potência e fatorial de um número inteiro (usar um algoritmo recursivo para cálculo do fatorial).
- Altere o menu definido no ponto 5 para incluir as novas operações (sugestão: quando o programa arranca pode ser perguntado ao utilizador se pretende usar uma calculadora científica ou não, sendo depois apresentado o menu de acordo com a escolha).
- No programa principal (método main da classe criada no package “calculos”), criar um objeto da classe “CalculadoraCientifica” que permita executar as operações existentes nesta classe.
- Desenvolver uma interface gráfica básica para o programa.


## Outros requisitos
- O código do programa deve estar dividido em classes;
- O código deve estar devidamente comentado;
- Caso seja utilizado código que não seja original do estudante, este deve estar claramente identificado e conter referência à fonte.
- É autorizada a utilização de ferramentas de Inteligência Artificial, desde que:
  - As prompts utilizadas sejam entregues juntamente com o projeto;
  - Seja claramente indicado que foi usado apoio de IA;
- O estudante compreenda e consiga explicar todo o código apresentado e efetuar alterações sobre esse código.
